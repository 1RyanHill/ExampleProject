Semantic HTML

Semantic elements are descriptive. Instead of simply using <div>, we can use <nav> to get a kind of division that tells a human reading the code what the purpose of a section is. Semantic elements include:

  • <article>
  • <aside>
  • <details>
  • <figcaption>
  • <figure>
  • <footer>
  • <header>
  • <main>
  • <mark>
  • <nav>
  • <section>
  • <summary>
  • <time>

These items are covered in detail at https://www.w3schools.com/html/html5_semantic_elements.asp
